新しい拠点で警備PROというシステムを使うために、本社のサーバへどうつなぐかを決めたい。ベンダーから150万円（5年分）の提案が出ているが、これが妥当か。もっと安く、安全に、運用できる方法はないか。この資料では、いくつかの選択肢を コスト・運用負荷・安全性・警備PROとの相性 の4つの観点で比べ、おすすめのプランを提示する。

結論（先に答え）

  • 推奨：既存の本社ルータ／ファイアウォール（YAMAHA RTX または Fortigate）を活用し、新拠点に小型のVPNルータ（YAMAHA RTX830）を1台置く構成。これでベンダー提案150万円／5年を、おおむね 40〜70万円／5年 に圧縮できる見込み。
  • AnyDeskを毎日の業務で常用するのはおすすめしない。理由は (1) 同時5〜10名のライセンスが年30〜60万円規模、(2) 接続先となるPCを5〜10台も本社に用意することになる、(3) 2024年に大規模な情報漏洩事件があり、業務での常時利用にはリスクが残る。
  • USBメモリで機密ファイルを持ち運ぶ運用はやめるべき。代わりに、VPN経由で本社のファイルサーバへ直接アクセスする運用に切り替え、誰がいつ何を見たかのログを取れるようにする。警備業法と全国警備業協会のガイドラインの観点でも望ましい。

詳しい検討

1. そもそも警備PROはどんなシステムか

警備PROはビジコン・ジャパン社（以下、ベンダー）の警備業務管理システム。公式サイトによれば、利用方法が2種類ある。

  • クラウド版：本社にサーバを置かず、ベンダーのクラウドを使う。テレワークや複数拠点に向いている、とベンダー自身が案内している。
  • オンプレミス版：自社内にサーバを設置し、各PCにクライアントソフトを入れて使う。社内ネットワーク内での運用を想定。

今回の前提は「本社にサーバがあるオンプレミス版を、新拠点5〜10台のPCからも使いたい」。ところが、オンプレミス版を離れた拠点から使えるか、ライセンスはどういう数え方なのか、推奨される接続方法は何か といった肝心の情報が、ベンダーの公開資料には載っていない。価格・ライセンス・動作環境はすべて「販売店に問い合わせ」になる仕組み。

そのため、最終的なネットワーク構成を決める前に、必ずベンダーに次の点を確認すべき。

  1. 新拠点のPCから本社サーバへ、インターネット経由（VPN利用）で接続する構成はサポートされるか
  2. 同時に何人接続するならライセンスは何本必要か（人単位／PC単位／同時接続単位など）
  3. 推奨されるOSやデータベースのバージョン、ネットワーク条件（応答速度や帯域の目安）
  4. クラウド版に切り替えた場合の月額・初期費用の見積

「実はクラウド版に切り替えた方が安いし手間も少ない」という結論になる可能性もあるため、4. のクラウド版見積は必ず取ることを強く推奨する。

2. ベンダー提案150万円／5年の中身を分解する

月額換算で約2.5万円／月。一般的にこの種の提案の内訳は次のような構成が多い。

  • 本社側ルータ／UTMの新規入れ替え：月1.0〜1.2万円相当（5年リース）
  • 新拠点側ルータ：含む
  • 設定・初期構築費：初期費用または月額に分散
  • 月額の保守（リモート監視・障害対応・設定変更代行）：月1.0〜1.3万円相当

自社で巻き取れる余地は大きい。

  • (a) 本社に既設のYAMAHA RTXやFortigateがあるなら、それを活用すれば「本社ルータ入れ替え」の費用は不要
  • (b) リースではなく機器を買い切りにすれば、5年で見た総額は安くなりがち
  • (c) 保守は「24時間監視」ではなく「年1回点検＋障害時のスポット対応」に絞れば月額は大幅に下げられる
  • (d) 拠点側のルータも、ヤマハRTX830など実績のある定番機種なら新品6万円程度

→ これらを組み合わせると、5年TCOは40〜70万円に収まる見込み（後述）。

3. AnyDeskを毎日の業務に使うのが向かない理由

AnyDeskは「自宅から会社のPCを遠隔操作する」用途では便利だが、新拠点の5〜10名が毎日業務で常用する用途には次のような問題がある。

ライセンス費用が見た目より高い

  • 個人向けプラン（Solo）は月14.90ドル前後だが、これは1ユーザー・同時接続1のみ
  • 5〜10名が同時に使うには上位プラン（Standard 月49.90ドル）＋同時接続アドオン（4〜9本）が必要
  • 合計 年30〜60万円 規模になる

接続先PCを物理的に用意する必要がある

  • AnyDeskは「あるPCを別のPCから操作する」仕組み。1つのPCに同時に複数人ログインする方式（Windows ServerのRDS方式）は正式サポートされない
  • つまり、5〜10名が同時に本社PCを操作するには、本社に PCを5〜10台用意 する必要がある（電気代・場所・OSライセンス・故障対応も別途）

セキュリティ事件が発生している

  • 2024年2月、AnyDesk社の本番システムが侵入され、コード署名証明書を全失効する事態になった
  • 直後、ダークウェブで AnyDesk顧客アカウント18,317件が15,000ドルで販売された
  • 漏洩アカウントの多くは二要素認証が未設定だったと報告されている
  • 業務で使うなら、二要素認証必須化、最新版維持、利用ログ監視など追加対策が前提

代替候補

  • リモートデスクトップ系で見るなら Splashtop Business（年15,000円/ID）が割安。10IDで年15万円程度
  • ただしどちらにせよ「接続先のPCを本社に置く」前提なので、根本的な構成上の課題は残る

セキュリティ強化策（同時に実施したい）

  1. VPNの暗号化方式は AES-256 など強度の高いものを選ぶ
  2. 通信ログを取得し、社内のサーバに集めて見られるようにする
  3. PCにはウイルス対策ソフトを集中管理で導入
  4. USBメモリは原則使えないように設定し、必要なときだけ会社管理の暗号化USBを許可
  5. ファイルサーバの「誰が何を見たか」のログを記録する

回線が止まったときの備え（BCP）

  • RTX830 は USB に LTEモバイル回線のドングルを挿してバックアップ回線にする運用が可能
  • 拠点回線が一時的に落ちても、モバイル経由で最低限の業務を継続できる

簡単な解説

  • B（ベンダー提案）：丸投げできるのが利点だが、コストが高い。月額の中身が「定期点検＋障害対応」なのか「設定変更代行のみ」なのかで自社で巻き取れる範囲が決まるので、まず内訳の文書化を求めるべき。

  • D（リモートデスクトップ）：本社にWindowsサーバを新設し、新拠点PCはその画面を遠隔操作する形。警備PROの動作は最も安定するが、初期投資が大きい（50〜100万円）。

6. USBメモリ運用は今のうちに見直す

「機密ファイルをUSBメモリで持ち運んでいる」運用は、近年の事故事例を見ても看過できないリスクがある。

過去の実際の事故例

  • 自治体で全市民約46万人分の個人情報入りUSBを紛失（2022年）
  • 企業で約928万件の顧客情報をUSBに持ち出し、名簿業者へ売却（内部不正）

警備業ならではの観点

  • 警備業は警備業法と全国警備業協会の「警備業における個人情報の保護に関するガイドライン」に沿った個人情報管理が求められる
  • 顧客現場の情報、隊員の個人情報、勤務記録などはとくに保護の必要性が高い
  • 1,000人を超える個人情報が漏洩した場合、個人情報保護委員会への報告義務もあり得る

改善のステップ

  1. VPN経由で本社のファイルサーバに直接アクセスする運用に切り替える
  2. ファイルサーバの権限を整理する：誰がどのフォルダを見られるかを部門別・機密度別に分ける
  3. ログを取る：誰がいつどのファイルを開いたかを記録できるようにする
  4. USBメモリは原則禁止に設定する：必要なときだけ会社管理の暗号化USBを許可する
  5. クラウドストレージとの併用も検討：写真や資料の共有は Microsoft 365 や Google Workspace、警備PRO関連の機密データは本社ファイルサーバ＋VPN、というように使い分ける

ビジコン・ジャパンの販売窓口に、次の4点を文書で確認する。

  3. 推奨される動作環境（OS、ネットワーク条件）
  4. クラウド版に切り替えた場合の見積（重要）

→ もしベンダーが「オンプレ版のVPN越え利用はサポート外、クラウド版を使ってほしい」と回答した場合、本件はクラウド版移行が最有力解になる。VPN投資自体が大きく不要になる可能性がある。
