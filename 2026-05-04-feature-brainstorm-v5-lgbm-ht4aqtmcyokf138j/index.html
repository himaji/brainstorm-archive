戦略的観点 (memory からの強い制約)

  1. overlay 戦略が本質 — 控除率 25% を越えるには「市場が見落としているもの」を入れる必要がある
  2. 市場視点の機械化は罠 — 上がり3F race-内偏差のように手動指標を綺麗に再現すると AUC は上がるが picks が本命寄りに収束して overlay が死ぬ
  3. 混合も壊れる — 市場視点 (dev3F) と非市場視点 (BW) を同時投入すると市場同調効果が overlay 検出を相殺
  4. 本物は地味に現れる — クリーンに +0.165 出る変更ほど artifact

→ 探すべきは「市場が処理しきれていない / 手間で人間が落としている / 馬券者の集合知に乗りにくい」情報。当日の馬体重 (騎乗前にしか見れない) / 母父 / parent stallion の長距離信頼性 / 脚質 × 馬場のクロスなどはこの方向に近い。逆に「上がり3F偏差」「直前オッズ推移」は市場と同じ穴を覗く方向で危険。
